# Formatting text to look nicer

**only works with version 4.12+**

`#` title (black background + white foreground)<br>
`:` orange<br>
`.` white<br>
`>` note (red)<br>
`*` note<br>
`-` note<br>
`|` highlight (yellow)<br>
`/` yellow<br>
`,` small text<br>
<br>
These only work at the start of a line, e.g.<br>
`#This is my title`
